# MCP Gateway Code Samples

## Description

The following folders contain code samples related to the MCP Gateway on NetApp Instaclustr. These samples show how to connect an AI agent to your own data and services through a single, secured [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) endpoint, exposing different backends such as HTTP servers and Apache Kafka as MCP tools.
