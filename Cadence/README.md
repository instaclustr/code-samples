# Cadence Code Samples

## Description

The following folders contain code samples related to [Uber Cadence](https://cadenceworkflow.io/) on NetApp Instaclustr. These samples demonstrate workflow orchestration patterns and integration with Apache Kafka.

## Projects

| Project | Description |
|---------|-------------|
| [CadenceKafka](./CadenceKafka/) | Demonstrates Cadence-Kafka integration using signals and activity completions |
| [CadenceKafkaDroneDeliveryDemo](./CadenceKafkaDroneDeliveryDemo/) | Full drone delivery demo app using Cadence workflows and Kafka messaging |

## Getting Started

Each sample has its own README that describes the sample, any prerequisites, examples, and anything else relevant to the code.

### Prerequisites

- Java 8 or higher
- Apache Maven
- Access to Apache Kafka and Cadence clusters (e.g., [Instaclustr managed services](https://www.instaclustr.com/))
