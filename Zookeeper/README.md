# Apache ZooKeeper Code Samples

## Description

The following folders contain code samples related to [Apache ZooKeeper](https://zookeeper.apache.org/) on NetApp Instaclustr. These samples demonstrate distributed coordination and synchronization patterns using ZooKeeper.

## Projects

| Project | Description |
|---------|-------------|
| [zookeeper-meets-Dining-Philosophers](./zookeeper-meets-Dining-Philosophers/) | Solves the classic Dining Philosophers problem using ZooKeeper distributed locks |
